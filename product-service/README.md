**1. Nacos 服务注册与发现**

模块名称：`order-service`、`product-service`、`user-service`

模块描述：  
本模块学习如何使用 Spring Cloud Alibaba 集成 Nacos 进行服务注册与发现。Nacos 是一个易于使用的动态服务发现、配置管理与服务管理平台。在本模块中，学习如何使用 Nacos 将微服务注册到注册中心，并使用 RestTemplate 实现服务之间的调用。
